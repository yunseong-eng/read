발표 시작하겠습니다.

저의 프로젝트명은 Running Crew입니다. 평소 저의 취미인 달리기와 관련된 프로젝트를 한번 만들어보고 싶어서 이 프로젝트를 진행하게 됐습니다. 먼저 두 개의 주요 DB 테이블을 사용합니다. 유저 테이블과 보드 테이블이 있는데 Users 테이블은 회원가입과 로그인 데이터를 처리하고, Board 테이블은 작성된 글의 데이터를 처리합니다. 

먼저, 메인 페이지입니다. 여기서는 HTML5의 Video 태그를 이용해 동영상을 배경으로 추가했습니다. 화면 크기에 따라 크기 자동 조절되게 만들어봤습니다. 처음에는 동영상이 전체적으로 나오지 않고 부분만 나와 전체적으로 나오게 맞추는 데 어려움이 있었지만, CSS의 object-fit 속성을 사용해 원하는 크기로 조절할 수 있었습니다. 메인 페이지 하단에는 Bootstrap 이용해 네 개의 카드를 배치했습니다. 이 카드를 통해 달리기와 관련된 정보들을 적어 꾸며봤습니다.

이제 구현한 기능들을 보여드리겠습니다.

첫 번째로, 회원가입 페이지입니다.  회원 가입 페이지의 배경에 사진을 흐릿하게 해서 깔았습니다. 그리고 JavaScript를 이용해 입력 검증 기능을 구현했습니다. 아무것도 입력하지 않은 상태에서 회원가입 버튼을 누르면, JavaScript(이벤트 리스너)를 통해 입력하라는 문구가 표시되며 포커스가 자동으로 해당 입력란으로 이동하게 만들었습니다. 회원가입을 해보겠습니다. 회원가입이 완료되면, (JSP와 서블릿을 이용해 데이터를 처리한 후) 메인 화면으로 이동하며, "회원가입이 완료되었습니다"라는 메시지가 나타납니다. <다시 회원가입 페이지로> 또한, (AJAX 요청을 통해 )중복된 아이디를 실시간으로 확인하고, 중복된 경우 "중복된 아이디입니다"라는 문구가 표시되도록 구현했습니다.

다음으로, 글 작성 부분을 보여드리겠습니다. 로그인하지 않은 상태에서 "글 작성하기" 버튼을 누르면 (서블릿 필터를 사용해) 로그인 페이지로 이동하게 됩니다.  아까 회원 가입 정보로 로그인을 해보겠습니다. 로그인 창도 회원가입과 동일하게 입력문구와 포커스 기능을 추가했고,로그인 과정에서는 (JSP와 서블릿을 통해 서버에서 아이디와 비밀번호를 확인하며,) 잘못된 정보가 입력되면 (JavaScript를 이용해) 오류 메시지가 표시됩니다. 올바른 정보를 입력하면 로그인에 성공하며, (세션을 통해 사용자 정보를 유지한 후) 메인 페이지로 돌아가게 됩니다.

로그인 후 "글 작성하기" 버튼을 누르면 크루 등록 페이지로 이동하며, 이 페이지에서는 핸드폰 번호와 사는 지역을 입력하도록 되어 있습니다.  (입력된 정보는 서블릿을 통해 데이터베이스에 저장됩니다.)  모든 페이지에는 (Anchor 태그를 이용해) 메인 화면으로 돌아갈 수 있는 링크를 기본적으로 포함시켰습니다.

핸드폰 번호와 사는 지역을 입력하면, 등록된 크루원들을 볼 수 있는 게시판 페이지로 이동합니다. 여기서는( JSP와 **JSTL (JavaServer Pages Standard Tag Library)**을 이용해 ) 데이터베이스에 저장된 정보를 동적으로 가져와 저장해둔 데이터와 방금 입력한 데이터가 화면에 표시하게 됩니다,( 이 과정에서 DAO 패턴을 사용해 데이터베이스와의 상호작용을 관리합니다.)

다시 메인화면으로 돌아가서 로그아웃 을 누르면 로그아웃이 완료되게끔 구현했습니다.

제가 구현한 기능과 프론트 단은 이 정도입니다. 원래 계획은 더 거창했지만, 실제로 만들다 보니 예상보다 더 많은 어려움이 있었습니다. 하지만 이 프로젝트를 통해 웹 개발의 흐름과 기능적인 부분을 많이 배울 수 있었습니다.

이만 발표를 마치겠습니다. 감사합니다.
