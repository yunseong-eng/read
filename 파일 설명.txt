BoardDAO.java: 데이터베이스에 연결하여 게시글을 삽입하거나 관련 데이터를 처리하는 DAO 클래스입니다.

BoardDTO.java: 게시판 데이터를 저장하고 전송하기 위한 객체를 정의한 클래스입니다.

UserDAO.java: 데이터베이스에서 사용자 정보를 삽입, 조회, 확인하는 DAO 클래스입니다.

UserDTO.java: 사용자 정보를 저장하고 전송하기 위한 객체를 정의한 클래스입니다.

body: 웹 페이지 전체의 기본 스타일을 설정합니다.
.login-background, .signup-background: 로그인 및 회원가입 페이지의 배경 이미지를 설정하고, 배경을 고정시키고 중앙에 배치합니다.
.centered-form: 폼 요소를 화면 중앙에 배치하고, 반투명 배경과 둥근 모서리를 설정합니다.
input, textarea: 사용자 입력 필드의 스타일을 지정합니다.
button: 버튼의 색상, 크기, 둥근 모서리 및 커서 스타일을 설정합니다.
header: 페이지 상단의 헤더 스타일을 정의합니다.

run.js 파일 분석
이 JavaScript 파일은 주로 회원가입과 로그인 폼의 제출을 처리하고, 서버와의 AJAX 통신을 관리하는 기능을 포함하고 있습니다.

board.jsp 파일 분석
이 JSP 파일은 웹 애플리케이션의 게시판 페이지를 구성합니다. 데이터베이스에서 게시판의 데이터를 가져와 HTML 테이블 형식으로 출력하는 역할을 합니다.

checkId.jsp 파일 분석
이 JSP 파일은 회원가입 시 아이디 중복 체크를 수행하는 역할을 합니다. 사용자가 입력한 아이디가 이미 데이터베이스에 존재하는지 확인하고, 그 결과를 클라이언트에게 반환합니다.

index.jsp 파일 분석
이 JSP 파일은 메인 페이지를 구성합니다. 사용자가 로그인되어 있는지 확인하고, 로그인된 사용자의 환영 메시지를 표시하거나, 로그인 페이지로 리다이렉트합니다.

login.jsp 파일 분석
이 JSP 파일은 사용자가 입력한 아이디와 비밀번호를 확인하여 로그인을 처리하는 역할을 합니다. 입력된 정보가 유효하면 사용자를 로그인시키고, 그렇지 않으면 오류 메시지를 반환합니다.

logout.jsp 파일 분석
이 JSP 파일은 로그아웃을 처리하는 역할을 합니다. 사용자의 세션을 무효화하여 로그아웃을 수행하고, 이후 메인 페이지 또는 로그인 페이지로 리다이렉트합니다.

signup.jsp 파일 분석
이 JSP 파일은 사용자가 입력한 정보를 바탕으로 회원가입을 처리하는 역할을 합니다. 입력된 정보가 유효하면 사용자를 데이터베이스에 저장하고, 그렇지 않으면 오류 메시지를 반환합니다.

write.jsp 파일 분석
이 JSP 파일은 사용자가 입력한 정보를 바탕으로 글쓰기 기능을 처리하는 역할을 합니다. 사용자가 핸드폰 번호와 사는 지역을 입력하여 글을 작성하고, 그 정보를 데이터베이스에 저장한 후 게시판 페이지로 리다이렉트합니다.

board.html 파일 분석
이 HTML 파일은 게시판 페이지를 구성하며, 사용자가 게시글을 작성하고 게시글 목록을 확인할 수 있는 인터페이스를 제공합니다.

index.html 파일 분석
이 HTML 파일은 웹 애플리케이션의 메인 페이지로, 사용자에게 런닝 크루 관련 정보와 비디오를 제공하며, 로그인, 로그아웃, 회원가입 등의 네비게이션 링크를 포함하고 있습니다.

login.html 파일 분석
이 HTML 파일은 로그인 페이지를 구성하며, 사용자가 아이디와 비밀번호를 입력하여 로그인할 수 있는 인터페이스를 제공합니다.

signup.html 파일 분석
이 HTML 파일은 회원가입 페이지를 구성하며, 사용자가 아이디, 비밀번호, 이름, 이메일을 입력하여 회원가입할 수 있는 인터페이스를 제공합니다.






BoardDAO.java:

데이터베이스는 정보를 저장하는 곳이에요. 이 파일은 데이터베이스에 연결해서 게시글(글)을 추가하거나 수정하는 일을 해요.
BoardDTO.java:

이 파일은 게시글(글) 정보를 저장하고, 다른 곳으로 보낼 때 사용해요.
UserDAO.java:

이 파일은 데이터베이스에 사용자 정보를 저장하거나, 사용자 정보를 가져오는 일을 해요.
UserDTO.java:

이 파일은 사용자 정보를 저장하고, 다른 곳으로 보낼 때 사용해요.
CSS 파일 분석
body:

웹 페이지 전체의 기본 디자인을 설정해요. 글꼴이나 여백 같은 걸 정해요.
.login-background, .signup-background:

로그인이나 회원가입 화면의 배경 이미지를 설정하고, 중앙에 오도록 배치해요.
.centered-form:

입력 창 같은 폼을 화면 가운데에 놓고, 배경을 살짝 투명하게 만들고, 모서리를 둥글게 만들어줘요.
input, textarea:

사용자가 글씨를 입력하는 창의 디자인을 정해요.
button:

버튼의 색깔, 크기, 모양을 정해요.
header:

웹 페이지 맨 위에 있는 제목 부분의 디자인을 정해요.
JavaScript 파일 분석
run.js 파일:
이 파일은 회원가입과 로그인할 때 사용자가 입력한 정보를 확인하고, 서버(컴퓨터)에 보내는 일을 해요.
JSP 파일 분석
board.jsp 파일:

이 파일은 게시판 페이지를 만들어요. 데이터베이스에서 게시글 정보를 가져와서 화면에 보여줘요.
checkId.jsp 파일:

회원가입할 때, 사용자가 입력한 아이디가 이미 사용 중인지 확인해요.
index.jsp 파일:

메인 페이지를 만들어요. 사용자가 로그인했는지 확인하고, 로그인했으면 환영 메시지를 보여줘요. 로그인이 안 되어 있으면 로그인 화면으로 보내요.
login.jsp 파일:

이 파일은 사용자가 입력한 아이디와 비밀번호를 확인해서 로그인해줘요. 아이디나 비밀번호가 틀리면 오류 메시지를 보여줘요.
logout.jsp 파일:

이 파일은 사용자가 로그아웃하게 해요. 로그아웃한 후에는 메인 페이지나 로그인 페이지로 보내요.
signup.jsp 파일:

회원가입을 처리해요. 사용자가 입력한 정보를 데이터베이스에 저장해요.
write.jsp 파일:

사용자가 글을 쓰고, 그 글을 데이터베이스에 저장한 후 게시판 페이지로 보내요.
HTML 파일 분석
board.html 파일:

게시판 페이지를 만들어요. 사용자가 글을 쓰고, 글 목록을 볼 수 있게 해줘요.
index.html 파일:

메인 페이지를 만들어요. 런닝 크루에 대한 정보와 비디오를 보여주고, 로그인, 로그아웃, 회원가입 버튼을 넣어요.
login.html 파일:

로그인 페이지를 만들어요. 사용자가 아이디와 비밀번호를 입력해서 로그인할 수 있게 해줘요.
signup.html 파일:

회원가입 페이지를 만들어요. 사용자가 아이디, 비밀번호, 이름, 이메일을 입력해서 회원가입할 수 있게 해줘요.
